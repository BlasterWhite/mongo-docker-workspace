# mongo-docker-workspace

## Description

This is a docker-compose project to create a MongoDB workspace with a MongoDB server and a MongoDB client.

## Usage

### Start the workspace

```bash
docker-compose up -d
```

### Stop the workspace

```bash
docker-compose down
```

### Default

- MongoDB server

  - Username: admin
  - Password: pass

- MongoDB client
  - host: localhost:8081
